# Wild Jacks - Gratis Online

**Wild Jacks / Sequence** – det klassiske kortspil med 10x10-bræt, chips og vilde Jacks!

✅ Spil mod AI  
✅ Online multiplayer med venner (snart!)  
✅ 100% gratis & open source (MIT license)  
✅ Lavet med Flutter – virker på Android & iOS

**Hvordan spiller man?**  
Læg chips på brættet for at lave to sekvenser på 5 i træk.  
One-eyed Jack = fjern modstanders chip  
Two-eyed Jack = placer chip hvor som helst
